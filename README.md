\# Gnomeswell (Unity 2D)



A complete 2D mobile game developed independently using Unity and C#, focusing on gameplay mechanics, UI, and level design. Created as a structured learning project guided by a textbook curriculum.



\## Features

\- 2D gameplay mechanics

\- Player movement and interaction

\- Level design and progression

\- UI implementation

\- Animation integration



\## Tech Stack

\- Unity (2D)

\- C#

\- Git



\## Attribution \& Learning Context

This project was developed as a learning exercise based on the book:



\*\*"Mobile Game Development with Unity: Build Once, Deploy Anywhere"\*\*



Some example code structures were initially provided by the book and then implemented, integrated, and fully understood by me as part of the learning process. The project was completed independently for educational and portfolio purposes.

This project was developed as a solo project for learning and portfolio use.
